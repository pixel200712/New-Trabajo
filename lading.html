Sobre mí

¡Hola! Soy Axel, un joven de 18 años apasionado por la programación y la poesía. Acabo de terminar la preparatoria y estoy comenzando mi camino en el desarrollo web.

Me encanta crear páginas modernas y funcionales que ayuden a negocios y personas a brillar en el mundo digital. Cuando no estoy escribiendo código, me gusta escribir versos y descubrir nuevas formas de expresarme.

Estoy siempre abierto a nuevos proyectos, retos y aprendizajes. ¡si lo puedes imagener lo puedes crear!

Tecnologías que domino

HTML5

Lenguaje de marcado para estructurar páginas web. Es la base del contenido en internet.

CSS3

Lenguaje de estilos que da diseño visual y responsivo a las páginas web.

JavaScript

Lenguaje de programación que permite crear sitios dinámicos, interactivos y funcionales.

Python

Lenguaje versátil usado en web, IA, automatización y análisis de datos.

MySQL

Sistema de gestión de bases de datos relacional, ideal para almacenar información web.

Bootstrap

Framework de CSS que acelera el diseño responsivo con componentes prehechos.
